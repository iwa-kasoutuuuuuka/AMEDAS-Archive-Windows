# AMEDAS-Archive-Windows

「AMEDAS-Archive-Windows」は、気象庁の過去の気象データ（アメダス観測データ）をWindows端末内にローカル蓄積し、民間API等に依存することなく独自の気象統計や特異日解析、地域間の気候比較を行うことができる、高速かつ高機能なネイティブデスクトップアプリケーションです。

## 特徴 (Features)

*   **完全ローカル・オフライン動作**
    *   取得した気象データはローカルの SQLite データベースに蓄積され、オフラインでも超高速で分析・検索が可能です。
*   **美麗なネイティブUI (WinUI 3)**
    *   Windows 11 に最適化された Fluent Design と Mica エフェクトを採用。滑らかで直感的な操作感を提供します。
*   **多様な分析機能**
    *   **単一検索**: 任意の観測所と期間を指定し、平均気温・降水量・日照時間などを一覧表示。
    *   **地域比較**: 2つの観測所を選び、グラフ (Canvas) で直感的に気候の違いを比較・対比できます。
    *   **特異日解析**: 指定した日付の前後のデータを複数年にわたって集計し、特定の天候になりやすい日（特異日）を自動検出します。
*   **自己完結ポータブル版の提供**
    *   Windows App SDK をアプリ内に内包しているため、ランタイムの事前インストールが不要な `.exe` シングルファイルでポータブルに動作します。

## 動作要件 (System Requirements)

*   **OS**: Windows 11 または Windows 10 (バージョン 1809 以降)
*   **アーキテクチャ**: x64

## ダウンロードとインストール (How to run)

最新版のポータブル ZIP ファイルは、以下の **Releases** ページからダウンロードできます。
👉 [最新リリース (Releases) をダウンロードする](https://github.com/iwa-kasoutuuuuuka/AMEDAS-Archive-Windows/releases/latest)

1.  上記リンクから `AmedasArchiveWindows_Portable_v1.x.x.zip` をダウンロードします。
2.  任意のフォルダに解凍します。
3.  フォルダ内の `AmedasArchiveWindows.exe` をダブルクリックするだけで起動します。

## 主な操作方法と使い方 (Usage)

アプリを起動したら、まずは気象庁のサーバーから過去の気象データをダウンロード（同期）してローカルデータベースを構築する必要があります。

1. **データの同期**
   * 左側のメニューから **「データ同期・ストレージ管理」** を開きます。
   * 「全国一括同期を開始」ボタンを押すと、全47都道府県のアメダス観測データを過去指定年数分（デフォルト10年）まとめて自動ダウンロードします。
   * ※途中でやめたくなった場合は「同期を停止」ボタンでキャンセル可能です。そこまでに完了したデータは保存されます。
   * ※特定の都道府県だけを同期したい場合は、リストから個別に「同期」ボタンを押してください。
2. **分析と検索**
   * 同期が完了したら、「単一検索」や「地域比較」「特異日解析」のメニューから観測所を選ぶことで、オフラインで超高速にデータを分析できます。

## 注意事項 (Important Notes)

*   **データ同期にかかる時間**: 
    「全国一括同期」を実行した場合、47都道府県すべての観測所データ（数十年分）をダウンロードするため、インターネット回線速度によって**数十分～1時間以上**かかる場合があります。
*   **ストレージ容量**: 
    全国分のデータを長期間分ダウンロードすると、ローカルの SQLite データベースファイル（`amedas_archive.db`）が数百MBになることがあります。不要になった都道府県のデータは「データ同期・ストレージ管理」画面の「クリア」ボタンから削除して容量を節約できます。
